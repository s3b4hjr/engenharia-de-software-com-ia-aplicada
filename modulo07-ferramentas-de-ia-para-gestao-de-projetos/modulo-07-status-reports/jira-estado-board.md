### Por que esse estado importa

O status report gerado com IA neste módulo parte deste snapshot. Se os status do seu board forem diferentes, o output da IA vai divergir do exemplo da aula, o que é ótimo para explorar, mas pode gerar confusão se você estiver acompanhando a demo pela primeira vez.

### Como usar este snapshot

1. Para acompanhar a demo, use a tabela acima como dado fonte da Sprint 4 encerrada.
2. Se estiver com o Jira ao vivo na Sprint 5, mantenha esse estado. Ele representa a continuidade operacional do curso.
3. Se quiser reproduzir visualmente a demo antiga, configure manualmente a Sprint 4 conforme a tabela, mas isso é opcional.
4. Para a gravação atual, a recomendação é não voltar o board: o objetivo aqui é comparar três relatórios gerados a partir do mesmo fechamento de sprint.
