## Estado esperado: Backlog recém-importado

Neste módulo o board representa o ponto de partida do projeto RouteWise: o backlog foi importado, mas nenhum trabalho foi iniciado ainda.

### Checklist de configuração

- [ ] Projeto Scrum criado no Jira Cloud
- [ ] Arquivo `routewise-jira-import.csv` importado com sucesso
- [ ] Épicos visíveis no backlog (`[EPIC] Segurança e Redução de Sinistros`, `[EPIC] Manutenção Inteligente`)
- [ ] Histórias ligadas aos épicos corretos via Epic Link
- [ ] Sprint 1 visível no backlog, mas **não iniciado**
- [ ] Todas as issues com status **"A fazer"**
- [ ] Nenhum sprint ativo

### Como verificar

Abra o Backlog do seu projeto. Você deve ver as histórias agrupadas em "Sprint 1" (sem o botão "Ativo") e o backlog plano abaixo. Se alguma issue estiver com status diferente de "A fazer", clique na issue e redefina o status manualmente.

### Como importar o backlog

Consulte o arquivo `Como Importar o Backlog no Jira - Módulo 1.pdf` nesta mesma pasta para o passo a passo completo de importação.
